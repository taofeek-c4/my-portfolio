Project Overview

This cattle disease detection app leverages a Convolutional Neural Network (CNN), specifically MobileNetV2, to automatically classify cattle as Healthy, or infected with Mastitis, Foot and Mouth Disease (FMD), or Lumpy Skin Disease (LSD) based on image analysis. Designed for ease of use and fast diagnosis, the system aims to support farmers and veterinary professionals with early, reliable disease detection, helping reduce economic losses and improve livestock health.

Key Features

Real-time Prediction

The system provides instant disease classification results from uploaded cattle images, enabling timely intervention and improved herd health management.

Responsive Design

The application is built with a user-friendly, mobile-responsive interface, ensuring seamless access across devices for both farmers and veterinary professionals.

Performance Optimization

Optimized for speed and efficiency, the app delivers fast disease predictions using lightweight deep learning models like MobileNetV2, making it suitable even for low-resource environments.

Easy Integration

Designed with a flexible backend (FastAPI), the app can be easily integrated into existing livestock management systems or extended with APIs for future scalability.
