Project Overview

A modern charity donation platform built to simplify the process of giving. The frontend connects seamlessly with a backend service developed using .NET, which integrates with Paystack to securely handle payments and transaction records.

Key Features

Secure Online Payments

Integrated with Paystack API to process donations safely and efficiently, supporting multiple payment methods including cards, transfers, and USSD.

Donation Tracking

Tracks all donor transactions in real time, providing transparency and accountability for both donors and administrators.

Responsive Design

The website adapts beautifully across all devices, offering a seamless donation experience on mobile, tablet, and desktop.

Admin Management System

An intuitive admin dashboard built to manage donations, view reports, and monitor donor activities with ease.
